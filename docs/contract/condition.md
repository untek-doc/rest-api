Фильтрация
===

Условия фильтрации отправляются в виде GET-параметров в ключе `filter`.

Можно задать фильтрацию по значению атрибута:

```json
{
	"filter": {
		"parent_id": 12
	}
}
```

Или сразу по нескольким атрибутам:

```json
{
   	"filter": {
		"parent_id": 12,
   		"category_id": 21
	}
}
```

При этом, условия буду склеены оператором `AND`.
